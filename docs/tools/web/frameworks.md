## **ars0n**

- [https://github.com/darklotuskdb/ars0n](https://github.com/darklotuskdb/ars0n)

Framework de **recon completo y post-explotación en bug bounty**.

Hace:

- Subdomain enum
- Port scan (naabu)
- DNS resolvers
- Screenshots (aquatone)
- HTTP probe (httpx)
- Nuclei
- JS endpoint discovery
- Wayback + gau

Ideal si quieres tener una base modular y rápida de levantar.

## **KarmaV2**

- [https://github.com/deoxyssecurity/KarmaV2](https://github.com/deoxyssecurity/KarmaV2)

Recon automation tool creada para bug bounty y pentest.

Hace:

- Subdomain discovery
- DNS bruteforce + permutation
- Live host detection
- Wayback machine
- Gau/JS parsing
- Fuzzing
- Nuclei
- Screenshots

Tiene modo de "baja interacción" y "full recon"

## **Rengine**

- [https://github.com/yogeshojha/rengine](https://github.com/yogeshojha/rengine)

Recon con dashboard web. Perfecto si trabajas en equipo.

Hace:

- Subdomain enum (Subfinder, Amass)
- Screenshots (aquatone)
- Naabu, Nmap
- Nuclei
- ParamSpider
- Wayback Machine
- Web UI para visualizar todo el progreso
