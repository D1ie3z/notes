## ¿Qué es?


> `httpx` es una herramienta de ProjectDiscovery que permite hacer **enumeración masiva de hosts HTTP(S)** y obtener información útil para priorizar y atacar.

Yo le tiro ese alv

```bash
httpx -l ips.txt -cl -sc -location -favicon -title -tech-detect -ip -ports 80,443,8000,8443,8080 -probe-all-ips -o htpx_out.txt -follow-redirects
```
