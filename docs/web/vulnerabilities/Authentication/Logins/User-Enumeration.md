## ¿Por qué?

Inclusive para siento que está de más esta nota, pero NUNCA ES SUFICIENTE, y nunca se sabe a quien le puede servir.

User enumeration literalmente, enumeración de usuarios quiere decir enumerar usuarios validos de la aplicación mediante muchas formas.


## Mensajes de error / Comportamiento

Intenta ver que tipo de respuesta te da la wea que estás hackeando. Ejemplo si pones un admin y te pone `admin no existe`, pero si pones un jorge-dev y pone `contraseña incorrecta` o un error similar (eso lo vas a ir viendo con tu ojito hacker XD) pues ahí haces una enumeración controlada (por opsec) para ver que usuarios validos hay en el sistema.

Prueba ese tipo de comportamiento en:

- Reset password
- Registros
- Inicio de sesión
