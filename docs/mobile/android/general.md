## No sé donde poner esto

Cuando iniciamos una busqueda de vulnerabilidades, el mejor punto de inicio es el AndroidManifest.xml

Una app de Android se comprende por varios elementos, algunos de ellos son:

- Activity: se ejecuta en primer plano y tiene interfaz de usuario.
- Broadcast Receiver: se ejecuta en segundo plano para realizar una tarea mínima(principalmente comunicación).
- Servicio: ejecuta tareas de larga duración en segundo plano.
